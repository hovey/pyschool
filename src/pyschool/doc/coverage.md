# Coverage

* coverage.py on [GitHub](https://github.com/nedbat/coveragepy) and on [PyPI](https://pypi.org/project/coverage/)

```bash
$ pip install coverage
$ coverage run -m unittest
$ coverage report
$ coverage html
$ cd htmlcov/
$ open index.html
```
